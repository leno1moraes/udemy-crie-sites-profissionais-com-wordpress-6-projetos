Multilingual WordPress Forms

== Description ==

Translate [Caldera Forms](https://calderaforms.com/caldera-forms?utm_source=dotOrg&utm_medium=plugins&utm_campaign=caldera-forms-translate) fields.

What it does:

* Translate form field labels, descriptions, default values and option values for select, checkbox, toggle switch, radio and auto-complete fields.
* Adds a language picker field.
* Translate the message displayed after a successful form submission.

One form, all the languages you need.

* [Documentation](https://calderaforms.com/doc/caldera-forms-translation-getting-started?utm_source=dotOrg&utm_medium=plugins&utm_campaign=caldera-forms-translate)
* Requires Caldera Forms 1.4.4 or later.
* Now supports language switching via [WPML](https://wpml.org/)
